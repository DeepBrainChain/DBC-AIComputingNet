客户端节点
=======================================================
安装：
```
    tar -zxvf dbc-linux-client-0.3.7.3.tar.gz
    cd dbc_repo
    修改conf目录下的core.conf配置文件：
        net_type=mainnet
        magic_num=0xF1E1B0F9
        rest_ip=0.0.0.0
```
启动：
```
    ./dbc -d
```
